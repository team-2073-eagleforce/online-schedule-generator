TBA Match Schedule CSV Generator

📋 How it works: Enter your personal TBA API key and competition code. The tool will fetch match data directly from The Blue Alliance and generate a CSV file for download.
Get your API key from TBA Account Settings
Format: {year}{region}{eventcode} - Find event keys on TBA event pages
Example Event Keys:
• 2024casd - 2024 San Diego Regional
• 2024week1 - 2024 Week 1 events
• 2024cmptx - 2024 Championship (Texas)

Troubleshooting:
• If download fails, try refreshing the page
• For CORS errors, try opening this page in an incognito window
• Check that your API key is valid at TBA Account
